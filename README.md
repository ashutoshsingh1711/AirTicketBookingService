# BOOKING MICROSERVICE

## INSTALL DEPENDENCIES

    npm i express;
    npm i body-parser;
    npm i http-status-codes;
    npm i mysql2;
    npm i sequelize sequelize-cli;
    npm i nodemon;
    npm i dotenv;
    npm i morgan;

## SETUP SEQUELIZE

    npx sequelize init; // CREATE FOLDERS
    npx sequelize db:create  // CREATE DATABASE ON MYSQL
